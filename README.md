# Conversor de Sistemas Númericos

## Essa aplicação tem como o intuito de forncer a possibilidade de converter números de diferentes sistemas númericos.

---

### Para o desenvolvimento do projeto foram utilizadas as seguintes ferramentas:

 - VS Code.
 - HTML5;
 - CSS3;
 - JavaScript.
 
   
---

## Sistemas númericos:

 - Binário;
 - Octal;
 - Decimal;
 - Hexadecimal.
